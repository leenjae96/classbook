# 🏫 ClassbookApplication (출석 재적 관리 통계 어플리케이션)

> **350여 명 규모의 주일학교 학생 및 교사를 위한 실시간 출석부 및 통계 애플리케이션입니다.**
> 기존 수기 방식의 비효율성을 개선하고, 데이터 기반의 출석 통계를 제공하기 위해 기획부터 배포까지 주도적으로 개발했습니다.
